Losse verkenningen (ter referentie):

Richting A

Clean SaaS

Rustig, veel wit, dunne lijnen, één accentkleur. Knoppen krijgen duidelijke hiërarchie: primair valt op, destructief is apart en ingetogen. Stijl à la Linear / Vercel.

 Richting B

Compact Pro

Informatiedicht maar strak geordend. Twee kolommen: acties links, metadata + budget als vaste zijbalk rechts. Strakke tabel-look, mono-cijfers. Voor power-users die veel runs volgen.

 Richting C — verfijnd ✦

Calm Minimal v2

C, maar opgeruimd: één primaire actie (Start developing) zichtbaar, álle commando's én links samen in één rustig menu. Geen losse knoprijen meer. Dit is de aanbevolen richting.

 Richting C — origineel

Calm Minimal v1

De eerste versie met losse knoprijen onder "Commando's" en "Links". Bewaard ter vergelijking met de verfijnde versie hierboven.

 Referentie

Huidige situatie (baseline)

Het scherm zoals het er nu uitziet, getrouw nagebouwd uit FactoryDashboardViews.kt + sf-ui.css. Gebruik dit als ijkpunt om de richtingen tegen af te zetten.
Hoe dit werkt. Deze mockups zijn pure HTML/CSS en bijna 1:1 portbaar naar de echte frontend: de gekozen CSS wordt straks de nieuwe sf-ui.css, en waar de structuur verandert (bv. knoppen groeperen in primair / secundair / destructief) passen we de helper-functies in FactoryDashboardViews.kt aan. Kies eerst een richting; daarna maak ik er één doorklikbaar, fijngeslepen prototype van.
